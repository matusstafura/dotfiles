# dotfiles

Personal dotfiles and setup scripts.

Includes configurations for:
- zsh
- neovim
- tmux
- wezterm
- i3wm
- polybar
- rofi
- vimrc
- p10k

## Installation

### Packages first

- Fedora

```bash
./install-packages.sh
```

### Basic desktop setup
```bash
git clone git@github.com:matusstafura/dotfiles.git
./install.sh
```

For systemd services, enable them with:

```bash
./install-systemd.sh
```
